The OpenSandbox C# SDK provides secure, isolated execution environments with comprehensive file system access, command execution, and resource management capabilities.

## Installation

<CodeGroup>
```bash .NET CLI
dotnet add package Alibaba.OpenSandbox
```

```powershell Package Manager
Install-Package Alibaba.OpenSandbox
```
</CodeGroup>

## Core Classes

### Sandbox

Main class for creating and managing sandbox instances.

#### Static Methods

##### CreateAsync()

Create a new sandbox instance with the specified configuration.

```csharp
using OpenSandbox;
using OpenSandbox.Config;
using OpenSandbox.Core;

var config = new ConnectionConfig(new ConnectionConfigOptions
{
    Domain = "api.opensandbox.io",
    ApiKey = "your-api-key",
});

await using var sandbox = await Sandbox.CreateAsync(new SandboxCreateOptions
{
    ConnectionConfig = config,
    Image = "python:3.11",
    TimeoutSeconds = 30 * 60,
    Env = new Dictionary<string, string> { ["PYTHONPATH"] = "/workspace" },
    Resource = new Dictionary<string, string> { ["cpu"] = "2", ["memory"] = "4Gi" },
});
```

<ParamField path="options" type="SandboxCreateOptions" required>
  Configuration options for sandbox creation.

  <Expandable title="properties">
    <ParamField path="ConnectionConfig" type="ConnectionConfig" required>
      Connection configuration for API server.
    </ParamField>

    <ParamField path="Image" type="string" required>
      Docker image to use.
    </ParamField>

    <ParamField path="TimeoutSeconds" type="int" default="600">
      Automatic termination timeout (server-side TTL).
    </ParamField>

    <ParamField path="Entrypoint" type="string[]" default='["tail", "-f", "/dev/null"]'>
      Container entrypoint command.
    </ParamField>

    <ParamField path="Resource" type="Dictionary<string, string>" default='{"cpu": "1", "memory": "2Gi"}'>
      CPU and memory limits.
    </ParamField>

    <ParamField path="Env" type="Dictionary<string, string>">
      Environment variables.
    </ParamField>

    <ParamField path="Metadata" type="Dictionary<string, string>">
      Custom metadata tags.
    </ParamField>

    <ParamField path="NetworkPolicy" type="NetworkPolicy">
      Optional outbound network policy (egress).
    </ParamField>

    <ParamField path="Volumes" type="Volume[]">
      Optional storage mounts (Host / PVC, supports ReadOnly and SubPath).
    </ParamField>

    <ParamField path="Extensions" type="Dictionary<string, string>">
      Extra server-defined fields.
    </ParamField>

    <ParamField path="SkipHealthCheck" type="bool" default="false">
      Skip readiness checks (Running + health check).
    </ParamField>

    <ParamField path="HealthCheck" type="Func<Sandbox, Task<bool>>">
      Custom readiness check function.
    </ParamField>

    <ParamField path="ReadyTimeoutSeconds" type="int" default="30">
      Maximum time to wait for readiness.
    </ParamField>

    <ParamField path="HealthCheckPollingInterval" type="int" default="200">
      Poll interval while waiting (milliseconds).
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="return" type="Task<Sandbox>">
  Fully configured and ready Sandbox instance.
</ResponseField>

<ResponseField name="return" type="Task<Sandbox>">

</ResponseField>

##### KillAsync()

<ResponseField name="return" type="Task">

## Service Interfaces

### Commands

Command execution service for sandbox environments.

#### RunAsync()

Execute a shell command in the sandbox.

```csharp
using OpenSandbox.Models;

// Simple execution
var execution = await sandbox.Commands.RunAsync("echo 'Hello World'");
Console.WriteLine(execution.Logs.Stdout.FirstOrDefault()?.Text);

// With streaming handlers
var handlers = new ExecutionHandlers
{
    OnStdout = msg => { Console.WriteLine($"STDOUT: {msg.Text}"); return Task.CompletedTask; },
    OnStderr = msg => { Console.Error.WriteLine($"STDERR: {msg.Text}"); return Task.CompletedTask; },
    OnExecutionComplete = c => { Console.WriteLine($"Finished in {c.ExecutionTimeMs}ms"); return Task.CompletedTask; },
};

await sandbox.Commands.RunAsync(
    "for i in 1 2 3; do echo \"Count $i\"; sleep 0.2; done",
    handlers: handlers
);
```

<ParamField path="command" type="string" required>
  Shell command text to execute.
</ParamField>

<ParamField path="options" type="RunCommandOptions">
  <Expandable title="properties">
    <ParamField path="Background" type="bool">
      Run command in background mode.
    </ParamField>

    <ParamField path="WorkingDirectory" type="string">
      Working directory for command execution.
    </ParamField>

    <ParamField path="TimeoutSeconds" type="int">
      Command execution timeout.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField path="handlers" type="ExecutionHandlers">
  Optional handlers for streaming events.
</ParamField>

<ResponseField name="return" type="Task<Execution>">
  Execution handle representing the running command.
</ResponseField>

#### GetCommandStatusAsync()

Get the current running status for a command.

```csharp
var status = await sandbox.Commands.GetCommandStatusAsync(execution.Id!);

```

<ParamField path="executionId" type="string" required>
  Unique identifier of the execution to query.
</ParamField>

<ResponseField name="return" type="Task<CommandStatus>">
  CommandStatus describing running state and exit code.
</ResponseField>

#### GetBackgroundCommandLogsAsync()

Get background command logs (non-streamed).

```csharp

```

<ParamField path="executionId" type="string" required>
  Unique identifier of the execution to query.
</ParamField>

<ParamField path="cursor" type="int?">
  Optional line cursor for incremental reads.
</ParamField>

<ResponseField name="return" type="Task<CommandLogs>">
  CommandLogs containing raw output and latest cursor.
</ResponseField>

## Configuration

### ConnectionConfig

Connection configuration for API server communication.

```csharp
using OpenSandbox.Config;

// Basic configuration
var config = new ConnectionConfig(new ConnectionConfigOptions
{
    Domain = "api.opensandbox.io",
    ApiKey = "your-key",
    RequestTimeoutSeconds = 60,
});

// Advanced: custom headers
var config2 = new ConnectionConfig(new ConnectionConfigOptions
{
    Domain = "api.opensandbox.io",
    ApiKey = "your-key",
    Headers = new Dictionary<string, string>
    {
        ["X-Custom-Header"] = "value"
    },
});
```

<ParamField path="options" type="ConnectionConfigOptions" required>
  <Expandable title="properties">
    <ParamField path="ApiKey" type="string">
      API key for authentication (can also be set via `OPEN_SANDBOX_API_KEY` environment variable).
    </ParamField>

    <ParamField path="Domain" type="string" default="localhost:8080">
      Sandbox service domain (can also be set via `OPEN_SANDBOX_DOMAIN` environment variable).
    </ParamField>

    <ParamField path="Protocol" type="ConnectionProtocol" default="Http">
      HTTP protocol (Http/Https).
    </ParamField>

    <ParamField path="RequestTimeoutSeconds" type="int" default="30">
      Request timeout applied to SDK HTTP calls.
    </ParamField>

    <ParamField path="UseServerProxy" type="bool" default="false">
      Request server-proxied sandbox endpoint URLs.
    </ParamField>

    <ParamField path="Headers" type="Dictionary<string, string>">
      Extra headers applied to every request.
    </ParamField>
  </Expandable>
</ParamField>

## Diagnostics and Logging

The SDK uses `Microsoft.Extensions.Logging` abstractions.

```csharp
using Microsoft.Extensions.Logging;
using OpenSandbox.Config;

using var loggerFactory = LoggerFactory.Create(builder =>
{
    builder.SetMinimumLevel(LogLevel.Debug);
    builder.AddConsole();
});

var sandbox = await Sandbox.CreateAsync(new SandboxCreateOptions
{
    Image = "python:3.11",
    ConnectionConfig = new ConnectionConfig(),
    Diagnostics = new SdkDiagnosticsOptions
    {
        LoggerFactory = loggerFactory
    }
});
```

## Exception Handling

The SDK throws `SandboxException` and derived exceptions for various error conditions.

```csharp
try
{
    var execution = await sandbox.Commands.RunAsync("echo 'Hello Sandbox!'");
    Console.WriteLine(execution.Logs.Stdout.FirstOrDefault()?.Text);
}
catch (SandboxReadyTimeoutException)
{
    Console.Error.WriteLine("Sandbox did not become ready before the configured timeout.");
}
catch (SandboxApiException ex)
{
    Console.Error.WriteLine($"API Error: status={ex.StatusCode}, requestId={ex.RequestId}, message={ex.Message}");
}
catch (SandboxException ex)
{
    Console.Error.WriteLine($"Sandbox Error: [{ex.Error.Code}] {ex.Error.Message}");
}
```

## Complete Example

```csharp
using OpenSandbox;
using OpenSandbox.Config;
using OpenSandbox.Core;
using OpenSandbox.Models;

var config = new ConnectionConfig(new ConnectionConfigOptions
{
    Domain = "api.opensandbox.io",
    ApiKey = "your-api-key",
});

try
{
    await using var sandbox = await Sandbox.CreateAsync(new SandboxCreateOptions
    {
        ConnectionConfig = config,
        Image = "python:3.11",
        TimeoutSeconds = 30 * 60,
    });

    // Write a file
    await sandbox.Files.WriteFilesAsync(new[]
    {
        new WriteEntry { Path = "/tmp/script.py", Data = "print('Hello World')", Mode = 644 }
    });

    // Execute command with streaming
    var handlers = new ExecutionHandlers
    {
        OnStdout = msg => { Console.WriteLine($"Output: {msg.Text}"); return Task.CompletedTask; }
    };
    await sandbox.Commands.RunAsync("python /tmp/script.py", handlers: handlers);

    // Get info
    var info = await sandbox.GetInfoAsync();
    Console.WriteLine($"State: {info.Status.State}");

    // Cleanup
    await sandbox.KillAsync();
}
catch (SandboxException ex)
{
    Console.Error.WriteLine($"Sandbox Error: [{ex.Error.Code}] {ex.Error.Message}");
}
```
